## Question To Evaluate Learning

If you see `5 // 2`, `5 % 2`, and `5 ** 2`, what does each expression return, and why?

<details><summary>Answer</summary>

`5 // 2` returns `2` because floor division keeps the whole-number quotient. `5 % 2` returns `1` because that is the remainder after division. `5 ** 2` returns `25` because `**` means exponentiation.

</details>
